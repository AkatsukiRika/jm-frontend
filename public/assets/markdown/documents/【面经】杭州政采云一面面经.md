## 面试形式
电话面试，时间比较短，主要聊的是八股文和项目。  
面试时间：2023/07/13 15:00~15:30

## 面试流程
1. 自我介绍

2. 说一下你比较擅长的技术栈？  
答：Kotlin、Flutter、Jetpack Compose

3. 了解 Kotlin 的协程吗，简单讲讲？

4. 有解决过内存泄露相关问题吗？展开讲讲内存泄露 SDK 的原理？

5. 有做过性能优化相关的事情吗？举例讲讲？

6. 你做了 Flutter Web 的移植，移植里面有什么难点吗？

7. Dart 是单线程模型，它是怎么实现异步的，跟 Kotlin 的协程有什么区别？

8. 说说 Android 的 Handler 机制。

9. Handler 中，当一个线程的消息队列里没有消息时，它在干什么？
