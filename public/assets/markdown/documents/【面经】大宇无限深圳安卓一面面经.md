## 面试形式

牛客视频面试，有算法题，到算法题的时候剩余时间不多了，所以只说了思路，没有写代码实现。

面试时间：2025/10/27 19:30~20:30

## 面试流程

### 1. 自我介绍

这部分我提了自己平时有写个人项目并开源在 GitHub 上，面试官让我发了代码仓库的地址，进行了简单的浏览。

### 2. 围绕项目的问题

- 项目中 AI 模型的使用、本地模型如何优化性能、本地模型和线上模型各自有什么优势和劣势

- 项目中有使用到 OpenGL，具体是怎样使用的

- 项目中有使用到 Kotlin Multiplatform 跨平台，是怎样用的，UI 层是怎么做的

### 3. 技术问题

- Android 开发中一般接触的是应用层，你了解应用层的几个协议，有用过哪些？

- 上传、下载在你们公司项目中是用的什么协议，有自己实现过分片传输、断点续传吗？

- 了解哪些加密算法，公司项目中用的是哪种？

- 关于跨平台用过哪些方案，除了 Flutter 和 Compose 外，有用过 React Native 这种吗，各自有什么优势和劣势？

- 说说线程和进程的区别（这里回答我提到了实际工作中主要使用协程）

- Kotlin 协程中有哪些调度器，一般最常用的是哪种？

- Kotlin 协程如何实现并发？

- Kotlin 的泛型实际在编译时会被擦除，有没有遇到相关的问题，是如何解决的？

- 包体积优化上有什么经验？

- 关于 MVC、MVP、MVVM、MVI 等架构有什么见解，说说各自的优势和劣势（这里回答提到了 MVVM 和 MVI 配合 Flow）

- 说说 StateFlow 和 SharedFlow、冷流和热流的区别

### 4. 算法题

给一个字符串（例如 2551214234），将它拆分成 IP 地址，返回所有可能的情况集合。
