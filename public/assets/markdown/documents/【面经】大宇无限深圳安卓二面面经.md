### 3. 技术问题（八股文）

Java、多线程问题

- Java 单例如何实现

- 实现后如何加锁

- 加锁后，单例对象这个变量还需不需要加上 volatile 参数

- synchronized 用在静态方法和普通方法上有什么区别

- Kotlin 的协程是基于线程的吗，内部是如何实现调度的

- 线程池是怎样实现的，讲讲线程池的参数

- 线程池参数 keepAlive 具体通过什么方式实现

自定义 View 问题

- 自定义 View 是如何绘制的

- 自定义 View 的触摸事件是如何传递的，涉及哪几个方法

- measure 相关有一个函数需要传一个 32 位的整数参数，那个参数是如何定义的（记不清楚这个参数是什么了）

- 讲一个你做过的比较复杂的自定义 View，以及你是如何优化它的性能

图片问题

- Android 图片格式有哪些

- 图片占用的内存如何计算

- 为什么图片实际占用的内存要比上面计算出来的多出来很多

- Glide 内部是如何对图片进行缓存的

- 如果要你设计一套缓存策略，你会怎么设计

- Glide 是如何复用 Bitmap 内存的；不使用 Native 方法如何复用 Bitmap 内存

- Glide 怎么增加一种新的类型

- 沉浸式导航栏是如何实现的

- 项目中有做过什么内存方面的优化吗，是如何做的（讲了 MNN 的 Native 内存优化）
