# 需求文档

## Question Deck Creator

### 功能简介

逐个输入问题和答案，创建知识问答卡片题组供复习时使用

调用的服务端接口参考 `server.md` 文档的「问题卡片」模块

### UI 设计

**首行：**

**左侧**是一个下拉菜单，用于选择文件，每个文件对应着一套题组（列表内容调用服务端「文件列表」接口获得），选定后会自动定位到该题组的第 1 题（题组内容调用服务端「获取内容」接口，`data_only` 参数传 `false`）

**中间**是核心的交互控件，简单示意为：➖ ⬅️ `1/99` ➡️ ➕ ⬇️

➖ 短按删除当前题目，长按删除整个文件（弹出确认对话框，需二次确认）

⬅️ 前往上一道题

`1/99` 进度指示器，展示「当前题目 / 总题目数」，点击后展示一个对话框，上面以 6 x 6 格式展示所有的题目序号，点击即可快速跳转到指定题目

➡️ 前往下一道题

➕ 短按在当前题组末尾增加一道题目（总是在末尾增加，并跳转到末尾），长按弹出输入文件名的对话框，点击确认后创建新的题组文件

⬇️ 点击后，将当前题组以 JSON 格式下载到本地（调用服务端「获取内容」接口，且 `data_only` 参数传 `true`）

**右侧**是保存按钮，点击后调用服务端「更新内容」接口，把变更保存到云端

**问题区域：**

位于首行下方，是一个很大的多行文本框

**答案区域：**

位于问题区域下方，是一个很大的多行文本框；问题区域和答案区域一同占据首行剩余的所有大小；两个区域的大小比值为 1:1，且中间有一定高度的 Space.
